# نُقطة — منصة الولاء (الإصدار الرابع)

منتج متكامل من **٥ صفحات** بتجربة شبه حقيقية: تسجيل → اختيار باقة → دفع → لوحة تحكم
مع حفظ بيانات العميل، دعم لغتين، وضعَين للألوان، والعملة بالدينار البحريني.

```

  ├─ خطوة ١: الاسم، البريد، الجوال

  ├─ Banner "اختر باقتك لإكمال التسجيل"

  ├─ المجانية: شاشة تفعيل مباشر

  ↓ يحفظ الباقة كـ active

  ├─ ترحيب باسم المستخدم الفعلي
```

## 🌍 دعم اللغتين + تحويل الأرقام التلقائي

كل مرة تتبدّل اللغة:
1. `applyTranslations(lang)` يحدّث كل `data-i18n`
2. `convertAllNumbers(lang)` يمشي على كل النصوص في الصفحة:
   - **العربية**: كل `0-9` → `٠-٩`
   - **الإنجليزية**: كل `٠-٩` → `0-9`

هذا يعني أنك تكتب أي رقم في HTML بأي طريقة، وهو يظهر بالصيغة الصحيحة للغة المستخدم تلقائياً.

## 💾 بيانات المستخدم (localStorage)

**المفتاح**: `nuqta_user`

```js
{
  name: "فاطمة الخالدي",
  email: "fatima@example.com",
  phone: "+973...",
  interests: ["مطاعم", "سفر", "جمال وعناية"],

  points: 500,                  // الرصيد (يبدأ ٥٠٠)
  tier: "bronze",               // يُحسب من النقاط
  plan: "pro",                  // null | "free" | "pro" | "vip"
  billing: "monthly",           // null | "monthly" | "yearly"
  planActive: true,             // بعد الدفع

  joinedAt: "2026-04-21T16:00:00Z",
  transactions: []              // فارغة للحساب الجديد
}
```

### API الموحّد (`window.NuqtaUser`)

| الدالة | الوصف |
|---|---|
| `get()` | يُرجع المستخدم الحالي أو `null` |
| `create(data)` | ينشئ مستخدم جديد فاضي (٥٠٠ ترحيبية) |
| `update(patch)` | تحديث جزئي |
| `activatePlan(plan, billing)` | يفعّل الباقة بعد الدفع |
| `calculateTier(points)` | يحسب المستوى من النقاط |
| `firstName()` | الاسم الأول (للترحيب) |
| `isRegistered()` | هل يوجد مستخدم مسجّل؟ |
| `clear()` | تسجيل الخروج + مسح |

## 💰 العملة والتسعير

العملة الرسمية: **دينار بحريني (د.ب / BHD)**

| الباقة | الشهري | السنوي | التوفير |
|---|---|---|---|
| الأساسية | مجاناً | — | — |
| المميّزة | ٥ د.ب | ٤٨ د.ب | ١٢ د.ب |
| VIP | ١٥ د.ب | ١٤٤ د.ب | ٣٦ د.ب |

**معادلات الحاسبة**:
- نقاط شهرية = الإنفاق × مضاعف المستوى (١×، ١.٥×، ٢×، ٣×)
- قيمة الخصم = النقاط × ٠.٠١ (كل ١٠٠ نقطة = ١ د.ب)
- نطاق شريط الإنفاق: ١٠ إلى ٥٠٠ د.ب

## 🚀 التشغيل

```bash
cd loyalty-v2
python3 -m http.server 8000
```

افتح <http://localhost:8000>

> يجب استخدام سيرفر محلي (ليس `file://`) لأن CSS يستخدم `@import`.

### اختبار التدفق الكامل

1. افتح الصفحة الرئيسية → اضغط "ابدأ مجاناً"
2. أكمل التسجيل بأي اسم
3. اختر باقة (المميّزة مثلاً)
4. في صفحة الدفع، أدخل أي بطاقة وهمية (رقم ١٦ رقم، MM/YY، CVV)
5. بعد النجاح تصل للوحة التحكم باسمك + ٥٠٠ نقطة + شارة Premium

## 🧩 ملاحظات تقنية

- **لا مكتبات خارجية** (سوى Google Fonts)
- **٥ صفحات HTML** + **١٢ ملف JS** + **١٣ ملف CSS**
- **بدون وميض** عند تحميل الصفحة (inline script في `<head>`)
- **RTL/LTR** تلقائي عبر `html dir`
- **Accessibility**: `aria-label`، semantic HTML
- **الأداء**: IntersectionObserver للـ reveal والعدّاد، `passive: true` للـ scroll
- **استهلاك منخفض**: ~٢١٠ كيلوبايت للمشروع كاملاً
