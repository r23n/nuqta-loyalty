## 🌍 دعم اللغتين + تحويل الأرقام التلقائي

كل مرة تتبدّل اللغة:
1. `applyTranslations(lang)` يحدّث كل `data-i18n`
2. `convertAllNumbers(lang)` يمشي على كل النصوص في الصفحة:
   - **العربية**: كل `0-9` → `٠-٩`
   - **الإنجليزية**: كل `٠-٩` → `0-9`

هذا يعني أنك تكتب أي رقم في HTML بأي طريقة، وهو يظهر بالصيغة الصحيحة للغة المستخدم تلقائياً.

## 🛠️ التقنيات المستخدمة

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Supabase (Auth + PostgreSQL + RLS)
- **Hosting**: Netlify
- **Fonts**: Reem Kufi, Rubik, Amiri (Google Fonts)

---

## 🎨 التصميم

- لوحة ألوان فاخرة: **ذهبي × نبيذي × كريمي**
- تصميم Responsive كامل (جوّال / تابلت / كمبيوتر)
- Animations محسّنة للأداء (Intersection Observer + requestAnimationFrame)
- RTL/LTR تلقائي حسب اللغة
